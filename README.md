# FileExplorer

## Описание

Проект представляет собой **консольный файловый менеджер**

- **Готов к сборке** в исполняемый **JAR-файл**.
- **Точка входа** — класс `FileManager`.
- **Код документирован** с использованием **JavaDoc**.
- Программа **корректно работает** как при запуске из **командной строки**, так и из **интегрированной среды разработки (IDE)**.

---

## Функционал

- **`copy <файл> <директория>`** — копирует файл в указанную директорию.
  - **Поддержка пробелов:** Если имя файла или путь к директории содержат пробелы, их необходимо **заключать в двойные кавычки** ("). Например: `copy "имя файла.txt" "путь к/директории"`.
- **`cd <путь>`** — переходит в указанную директорию.
- **`exit`** — завершает работу программы.
- **Отображение размеров** файлов в удобочитаемом формате (B, KB, MB, GB).
- **Создание целевой директории**, если она не существует.
- **Добавление суффикса**, если файл с таким именем уже существует.
- **Защита от выхода выше корня диска**.
